# 詰将棋の読み上げツール

- Windowsで動作する詰将棋の読み上げるGUIツール
- 開発向けにはUbuntuなどのLinux環境もサポート
- GUI上でKIFファイルの入ったディレクトリを指定して、詰将棋の内容を音声で読み上げる
- WindowsとWebページの両方で動作することを考慮する

このようなKIFファイルが入力された場合

攻め方 よんよん　ぎん

と読み上げます。手数がある場合もありますが、手数は読み上げません。
将棋独特の符号の読み上げをサポートします。

- 例えば、"7六歩"は「ななろく ふ」と読み上げます。
- "5三銀"は「ごうさん ぎん」と読み上げます

## GUIの振る舞い

- フォルダ選択
- もし初回起動でなければ、前選択しておいたフォルダが読み込まれた状態になる
- フォルダの先頭のKIFファイルを読み込み、内容を読み上げる
- Spaceキーで次のKIFファイルを読み込む
- 右矢印で次のKIFファイルを読み込む
- ←左矢印で前のKIFファイルを読み込む
- Aキー: 攻め方の駒配置を読み上げる
- Sキー: 受け方の駒配置を読み上げる
- Dキー: 持ち駒を読み上げる
- Fキー: 読み上げのみで詰将棋を解くのをあきらめて、駒の配置を表示する

## 実装方式

- 音声はSpeechSynthesis APIを使用して読み上げる
- KIFファイルを解析して、必要なテキストを順番に読み上げる

## 開発方針

### 技術スタック

| 層 | 技術 | 理由 |
|---|---|---|
| デスクトップフレームワーク | Tauri v2 (Rust + Web) | 軽量（~10MB）、Windows/Linux対応、Webフロントエンド共用可 |
| フロントエンド | React + TypeScript + Vite | 高速ビルド、Web版との共通化が容易 |
| KIFパーサー | tsshogi (npm) | MIT、依存0、KIF/KI2/CSA/JKF/SFEN対応、活発に開発中 |
| 音声再生 | SpeechSynthesis | ブラウザ/Tauri両対応、追加ファイル不要 |
| 設定永続化 | tauri-plugin-store / localStorage | デスクトップ版はstore、Web版はlocalStorage |
